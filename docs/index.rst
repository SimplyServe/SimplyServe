SimplyServe Documentation
=========================

SimplyServe is a meal planning and nutrition-tracking application developed for SETaP Coursework Iteration 2.

The system supports authentication, recipe browsing, recipe creation and editing, meal planning, meal logging, allergy filtering, shopping list management, nutritional summaries, and a meal spinner for random recipe suggestions.

.. note::

   This documentation describes the implemented SimplyServe system, including the requirements, architecture, frontend, backend, API endpoints, development workflow, and testing approach.
