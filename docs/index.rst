SimplyServe Documentation
=========================

SimplyServe is a meal planning application developed by Team 2D for the SETaP Coursework Iteration 2. The application supports smart meal suggestions, allergen filtering, calorie coaching, nutritional tracking, recipe management, shopping lists, meal planning, and JWT-based authentication.

.. note::

   This documentation describes the implemented Iteration 2 system, including the frontend, backend, API endpoints, architecture, requirements, and testing approach.
